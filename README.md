# 間接引語闖關課件

互動式英文課件，主題為 Indirect Speech（間接引語）。

## 檔案說明

| 檔案 | 用途 |
|------|------|
| index.html | 入口頁面 |
| student.html | 學生端：三層關卡、填充/選擇/改寫、錯題本、成績記錄 |
| 	eacher.html | 教師端：題庫管理、數據看板、學生管理、CSV 匯出 |

## 使用方式

1. 學生開啟 student.html，輸入姓名開始闖關
2. 完成後按「匯出」按鈕，下載 JSON 作答記錄
3. 教師在 	eacher.html（PIN: 1234）中匯入學生 JSON
4. 查看數據看板與各項圖表分析
